# SpeedCalculator
Calulates the speed of a person real time
<details>
<summary> <b> Details </b></summary>
  
The projects purpuse is to make a website that measures the speed of your movement using Geolocation API and Haversine formula for calculating the distance. 
Then we can calculate the speed by the formula: Average Speed = Total Distance ÷ Total Time.
!!it wokrs best on mobile phone, and works best with larger speed.
